# Tesla Website Replication

This project is a full-stack replica of key features from the official Tesla website. It delivers a sleek UI, responsive layout, secure authentication, and a dynamic car customization experience similar to Tesla's product pages.

#Project Scope

The project includes multiple pages, a mobile-friendly navigation system, and interactive car customization with dynamic pricing.

# Pages Included

1. Home Page – Banner, hero section, CTA buttons

2. Cars / Products Page – Show at least three Tesla models

3. Customize Car Page – Options for:

   - Exterior color

   - Wheels

   - Interior

   - Dynamic price calculation

   - Live preview updates

4. Login / Signup Page – Secure user authentication

# Authentication Features

- Secure login/signup using hashed passwords (bcrypt)

- Session-based authentication (express-session)

- Protected routes for user info generation

- Input validation and error handling

# Frontend Features

- Built with React + Vite for performance and modern tooling.

- Clean Tesla-style design

- Responsive navigation

- Full-screen mobile menu

- Image or layered preview system for car customization

- Dynamic UI updates based on user selections

# Backend Features

- Backend powered by Node.js + Express.

- API endpoints for authentication

- Session management

- Request validation and error handling

# Database

- Uses MongoDB (MongoDB Atlas).

- Collections include:

users → Account info
